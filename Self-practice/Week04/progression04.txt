
ใช้ chat gpt ในการสร้างโจทย์และนำโจทย์มาฝึกทำตรงไหนที่ยังไม่ค่อยเข้าใจใช้ Ai ช่วยอธิบายในส่วนที่ตนเองยังไม่เข้าใจ ทำให้เข้าใจแต่ละ method มากขึ้น

filter(callback) ใช้กรองค่าใน array โดยจะคืนสมาชิกที่ผ่านเงื่อนไขใน callback
เช่น filter((c) => c.length <= 5) → เลือกค่าที่ความยาว ≤ 5

map(callback) ใช้แปลงค่าใน array ทุกตัว แล้วคืน array ใหม่ที่แปลงแล้ว
เช่น map((c) => c.toUpperCase()) → เปลี่ยนทุกคำให้เป็นตัวพิมพ์ใหญ่

includes(value) ใช้ตรวจสอบว่ามี “คำ/ค่า” นั้นอยู่หรือไม่
ถ้าเป็น string → เช็ค substring
ถ้าเป็น array → เช็คว่า array มี element นั้นไหม
เช่น "Mango".toLowerCase().includes("mango") → true

find(callback) ใช้หาค่าตัวแรกใน array ที่ตรงตามเงื่อนไข
ถ้าเจอ → คืนค่านั้น
ถ้าไม่เจอ → คืน undefined
เช่น find((f) => f.includes("orange"))

การใช้ .toLowerCase() / .toUpperCase() ทำให้ไม่สนใจตัวพิมพ์เล็ก/ใหญ่ (normalize ก่อนค้นหา/เปรียบเทียบ)

การใช้ % (modulus) ใช้เช็คเลขคู่/เลขคี่ เช่น id % 2 !== 0 หมายถึง “เลขคี่”

if ... else
ใช้ตรวจสอบค่าที่ได้จาก find() ว่ามีหรือไม่ (undefined หรือเจอจริง ๆ)
