สิ่งที่ผมได้เรียนรู้ด้วยตนเองเพิ่มขึ้นจาก Week01 

Array (อาร์เรย์)

-สร้างอาร์เรย์เก็บข้อมูลหลายชนิดได้ เช่น ตัวเลข, ข้อความ, Boolean
-เช็คความยาว .length
-เข้าถึงสมาชิกด้วยดัชนี [index] (เช่น ตัวแรก [0] ตัวสุดท้าย [length - 1])
-เพิ่มสมาชิกที่ต้นอาร์เรย์ .unshift()
-เอาสมาชิกตัวแรกออก .shift()
-กรองสมาชิกตามเงื่อนไข .filter()
-เช็คว่าค่ามีในอาร์เรย์หรือไม่ .includes()


Object (ออบเจ็กต์)

-สร้างออบเจ็กต์เก็บข้อมูลแบบเป็นกลุ่ม (property) เช่น brand, model, year
-ใส่ฟังก์ชันในออบเจ็กต์เรียกว่า method เช่น getCarInfo()
-ใช้ this อ้างถึงตัวออบเจ็กต์เองใน method
-เรียกดู property หรือ method ของออบเจ็กต์ด้วย object.property หรือ object.method()

สิ่งที่ผมได้เรียนรู้ด้วยตนเองเพิ่มขึ้นจาก Week02
-เข้าใจว่า object literal ถ้ามีคีย์ซ้ำ ค่าตัวท้ายจะทับตัวแรก
-ใช้ computed property name เพื่อสร้างคีย์แบบ dynamic ได้
-รู้จัก ternary operator สำหรับเขียนเงื่อนไขสั้นๆ
-การสร้าง class ด้วย constructor และการใช้ this อ้างถึงอินสแตนซ์
-method ของ class ถูกเก็บบน prototype และสามารถแก้ไขค่าของ property ได้

สิ่งที่ผมได้เรียนรู้ด้วยตนเองเพิ่มขึ้นจาก Week03
-เข้าใจการตรวจสอบค่าอินพุต (validation) ก่อนประมวลผล เช่น เช็คชนิด string และ trim ข้อความ
-เรียนรู้การใช้ toLowerCase() เพื่อลดปัญหาตัวพิมพ์ใหญ่เล็กไม่ตรงกัน
-รู้จักการใช้ Regular Expression ที่รองรับยูนิโค้ด (\p{L}, \p{N}) เพื่อแยกคำได้หลากหลายภาษา
-ใช้ Nullish Coalescing (??) ในการจัดการค่าที่อาจเป็น undefined เวลานับความถี่
-เข้าใจข้อจำกัดของ regex กับภาษาไทย และเห็นว่ามีทางเลือกอย่าง Intl.Segmenter สำหรับตัดคำที่แม่นยำขึ้น

สิ่งที่ผมได้เรียนรู้ด้วยตนเองเพิ่มขึ้นจาก Week04
-รู้จักการใช้ filter() เพื่อกรองข้อมูลที่ตรงเงื่อนไขออกมา
-ใช้ map() เพื่อเปลี่ยนค่าทุกตัวใน array ได้ง่ายๆ
-เข้าใจการประยุกต์ map() + filter() ร่วมกันเพื่อดึงข้อมูลเฉพาะส่วนที่ต้องการ
-ใช้ includes() เพื่อตรวจสอบว่าข้อความมีคำย่อย (substring) อยู่หรือไม่
-ใช้ find() เพื่อตรวจหาสมาชิกตัวแรกที่ตรงเงื่อนไข และเช็คว่ามีหรือไม่มีข้อมูลนั้น

สิ่งที่ผมได้เรียนรู้ด้วยตนเองเพิ่มขึ้นจาก Week05
-การเขียนฟังก์ชันพื้นฐานที่รับค่าเข้าแล้วส่งค่ากลับ
-การใช้ตัวดำเนินการ % เพื่อตรวจสอบเลขคู่หรือคี่
-การจัดการสตริงด้วยการแยกเป็นตัวอักษร กลับลำดับ และรวมกลับ รวมถึงการตรวจสตริงที่อ่านเหมือนกันจากหน้าหลัง
-การหาค่าต่ำสุดในอาร์เรย์ด้วย Math และ spread operator
-การใช้ Regular Expression เพื่อนับสระในข้อความ
-การจัดการอาร์เรย์ เช่น ลบค่าซ้ำด้วย Set หรือทำให้อาร์เรย์ซ้อนกันหลายชั้นแบนลงด้วย reduce และ recursion
